# Csv transactions file editor

### Used dependencies

```
    "bootstrap",
    "prop-types",
    "react",
    "react-bootstrap",
    "react-csv",
    "react-csv-reader",
    "react-dom",
    "react-redux",
    "react-router-dom",
    "redux"
```

### INSTRUCTIONS

1. Install all dependencies with command:
   `npm install`
2. Run project in your browser with command:
   `npm start`
